# Conference posters repository

This repository contains a collection of LaTeX posters presented at various conferences.

The posters' repositories can include:
- /_images: university logos/banners and QR codes
- /data: .csv databases for figure plotting
- /pics: .png or .pdf figures derived from data analyses
- /tikz: .tex tikz files for figure plotting
- /main.tex and /main.pdf: main LaTeX file and the compiled .pdf file
- /stefan.bst and uibkpost.cls: LaTeX
